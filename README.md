# todo list
- you can add new task - function: addTask(task, status, priority) status and priority has default values 'To Do', 'Low'
- change task status - function: changeStatus(task, newStatus)
- change task priority - function: changePriority(task, newPriority)
- delete task - function: deleteTask(task)
- view the entire list of tasks - function: Showlist()
