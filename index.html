Функции calc

Для вызова функции calc откройте консоль в браузере с помощью кнопки F12 (Crome).

Далее в консоли необходимо ввести вызов функции: calc(action, a, b), где:

  • action - действие. Доступные действия: add (+), multi (*), subtract (-)
  • a - произвольное число
  • b - произвольное число
